Process Flow:

1. User requests extraction from URL
   ↓
2. Check domain rate limiting (if Redis available)
   ↓
3. Try jsdom (FAST):
   - Generate random user-agent
   - Generate stealth headers
   - Add random delay (100-300ms)
   - Fetch HTML via axios
   - Check for block indicators
   - Parse with JSDOM
   - Apply 10-layer deobfuscation
   - Get confidence score
   ↓
4. If emails found → Return (success!)
   ↓
5. If no emails → Try Puppeteer (THOROUGH):
   - Generate random user-agent
   - Wait for browser slot
   - Add random delay (200-500ms)
   - Launch browser with stealth args
   - Apply navigator.webdriver spoofing
   - Navigate with stealth headers
   - Extract rendered content
   - Apply 10-layer deobfuscation
   - Get confidence score
   ↓
6. Return emails with confidence and method attribution
   ↓
7. Set domain cooldown (2-5 seconds)
   ↓
8. Log results with full details
