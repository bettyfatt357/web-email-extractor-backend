2. NEW FUNCTIONALITY
───────────────────────────────────────────────────────────────────────────────

INPUT: Advanced Search Request
  - keywords: ["hedge fund manager", "real estate agent"]
  - patterns: ["@company.com", "@startup.co"]
  - location: "Florida, USA"
  - searchDepth: 2
  - delayMs: 300
  - deepSearch: false

PROCESS:
  For each keyword:
    1. Combine keyword + location
    2. Enhance query (auto-detect context)
    3. Google search (N pages)
    4. Extract URLs
    5. Filter URLs (general filters)
    6. Filter by patterns (domain-specific)
    7. Queue jobs
    8. Sleep between keywords
    9. Track results by keyword

OUTPUT: Grouped Search Results
  {
    searchId, searchMode: "advanced", totalQueued: 28,
    jobsByKeyword: {
      "hedge fund manager": { jobCount: 15, jobIds: [...] },
      "real estate agent": { jobCount: 13, jobIds: [...] }
    }
  }
